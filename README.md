# server monitor（服务器监控程序）

可登陆管理自己的多台服务器，实现实时监控。
后端使用Django框架。

部分运行图可在该项目的source目录中下载。

### 使用方法：
首先导入 auto.sql 数据库或执行 create_monitor.sql 中的部分代码生成数据库。<br>
然后需要对 settings 文件进行数据库配置。<br>
如果是使用的 auto.sql 导入数据库，
